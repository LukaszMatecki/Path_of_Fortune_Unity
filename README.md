## Wersja Polska

### O Projekcie
**Path of Fortune** to hybryda gry planszowej i karcianki, stworzona w silniku Unity. Projekt łączy klasyczne mechaniki rzutu kostką z turowym systemem walki opartym na kartach oraz ekonomią gry. Celem było stworzenie żyjącego świata z własnym cyklem dobowym i immersyjną oprawą audiowizualną.

Gra demonstruje zaawansowaną obsługę UI, zarządzanie stanem gry (Save/Load) oraz integrację mechanik karcianych w środowisku 3D.

### Specyfikacja Funkcjonalna

| Moduł | Opis |
| :--- | :--- |
| **System Ruchu** | Fizyczny rzut kostką oraz nawigacja po waypointach na planszy. |
| **Walka Karciana** | Turowy system walki. Gracz używa talii kart do ataku i obrony przeciwko wrogom 3D. |
| **Ekonomia** | System zbierania monet ze skrzynek i wydawania ich na leczenie (Health Restoration). |
| **Cykl Dobowy** | Dynamiczny system Day/Night zmieniający oświetlenie i klimat sceny. |
| **Animacje** | Kompletny zestaw animacji dla postaci gracza, przeciwników i interakcji z otoczeniem. |
| **Core Systems** | Pełny system zapisu (Save/Load), lokalizacja (PL/EN) oraz Audio Manager. |

## English Version

### About the Project
**Path of Fortune** is a board game and card battler hybrid developed in Unity. The project blends classic dice-rolling mechanics with a turn-based card combat system and in-game economy. The goal was to create a living world featuring a day/night cycle and immersive audiovisuals.

The game showcases advanced UI handling, game state management (Save/Load), and card mechanic integration within a 3D environment.

### Key Features

* **Movement System:** Physics-based dice rolling and board navigation.
* **Card Battle System:** Enemy encounters trigger a card-based minigame. Players play cards from their hand to attack enemies or defend against incoming attacks.
* **Economy & Loot:**
    * Collect coins from chests scattered across the map.
    * Spend collected coins to restore player health.
* **Game World:**
    * **Day & Night Cycle:** Dynamic lighting system changing the game's atmosphere.
    * **Animations:** Extensive animation set for characters, enemies, and environment.
* **Core Systems:**
    * **Save & Load:** Complete system to save and resume game progress.
    * **Localization:** Language switching system (Polish/English).
    * **Audio:** Soundtrack and Sound Effects (SFX) implementation.

### Tech Stack
* **Engine:** Unity 6 (Version 6000.0.24f1)
* **Core Logic:** C# (Card logic, Combat, SaveSystem, Managers)
* **3D Assets:** Blender (Enemy models, chests, environment)
* **2D Art:** Photoshop / Illustrator (Cards, UI, icons)

### How to Run

1. Clone the repository:
   https://github.com/LukaszMatecki/Path-of-Fortune.git
2. Open in Unity Hub:
   Select the project folder (Required Unity version: 6000.0.24f1).
3. Play:
   Open MainScene.unity and press the Play button.
